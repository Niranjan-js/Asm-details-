📖 COMPLETE ARCHITECTURE EXPLANATION

🏗️ Overview

This ASM (Attack Surface Management) platform is a full-stack, enterprise-grade solution for continuous security monitoring. It follows a modular, async-first architecture with 15+ layers, 20+ open-source tools, and AI-powered risk analysis.

👥 User Layer

Three primary personas: Security Analysts (track vulnerabilities, generate reports), Pentesters (plan engagements using discovered assets), SOC Teams (monitor real-time dashboard). RBAC controls access per role.

🖥️ Frontend Layer

Built with Next.js 14 (React) providing dashboard, asset/scan management, findings viewer, and report generator. Communicates via REST API with FastAPI backend. Served through Nginx with real-time polling.

⚙️ Backend Layer

FastAPI (Python) - high-performance async framework with automatic OpenAPI docs. Provides Auth, Targets, Scan Jobs, Assets, Results, Reports, and Users APIs. Uses JWT + RBAC, SQLAlchemy ORM, Pydantic validation.

🎯 Scan Orchestrator

The brain of the platform - receives scan requests, decomposes into tool-level tasks, manages dependency ordering (Discovery → Enumeration → Vulnerability), handles rate limiting, retries, and status tracking.

⚡ Redis + Celery

Redis acts as message broker and result backend. Celery workers are stateless, horizontally scalable containers executing one tool per target. Supports auto-scaling via Docker Compose and task priorities.

🔍 Discovery Workers

Four tools run in parallel: Subfinder (passive via cert transparency), Amass (active+passive DNS), Assetfinder (cert logs), Findomain (fast CT-based). Results deduplicated for maximum coverage.

📡 Enumeration Workers

HTTPX (live host detection), Katana (active web crawling), GAU/Waybackurls (historical URLs), DNSX (DNS resolution), Nmap (port scanning). Builds complete asset inventory with services and technologies.

🛡️ Vulnerability Workers

Nuclei (9,000+ templates for CVEs/misconfigs), SSL Scanner (TLS analysis), Tech Detection (WhatWeb/Wappalyzer), CloudBrute/S3Scanner (cloud misconfigs). Stores findings with CVE IDs and evidence.

🗄️ Storage Layer

PostgreSQL as single source of truth (targets, assets, vulnerabilities, audit logs) with SQLAlchemy ORM. File storage (S3-compatible) for screenshots, raw outputs, and generated reports.

🧠 AI & Risk Engine

Post-scan intelligence: calculates 0-100 risk scores, prioritizes CVEs using EPSS/KEV, performs attack path analysis, tags asset criticality, generates AI remediation recommendations and executive summaries.

🔗 External Integrations

Fetches live threat intel from CVE/NVD/EPSS/KEV, VirusTotal (reputation), Shodan (internet scans), WhoisXML/IPInfo (geolocation/ASN). Slack/Email/Teams for real-time alerts. Async fetching never blocks scans.

⏱️ Monitoring & Scheduling

Cron-like scheduler for automated daily/weekly scans. Change detection compares new vs previous results. Continuous monitoring re-scans critical assets hourly. 24/7 visibility via live dashboard.

🔔 Alerting System

Multi-channel alerts: Email (SMTP), Slack/Teams (rich notifications), in-app center, webhook (SIEM integration). Configurable per-target rules for critical findings and scan completions.

📄 Reporting & Outputs

Professional PDF reports with executive summaries and vulnerability tables. CSV/JSON exports for raw data. Export API for automated ingestion. Supports SOC2, ISO 27001 compliance.

🚀 Infrastructure & Deployment

Fully containerized with Docker + Compose. Nginx reverse proxy for TLS termination. CI/CD via GitHub Actions. Prometheus+Grafana for metrics, Loki/ELK for centralized logging. Nightly backups to S3.

Trigger
Analyst creates scan job via Next.js → FastAPI validates and stores as PENDING
2
Orchestrate
Orchestrator builds task graph → pushes Discovery tasks to Redis queue
3
Discovery
Celery workers run Subfinder/Amass/etc in parallel → subdomains stored in PostgreSQL
4
Enumerate
HTTPX probes live hosts → Nmap scans ports → DNSX resolves records
5
Vulnerability Scan
Nuclei runs 9,000+ templates → SSL Scanner → Tech Detection → CloudBrute
6
AI Enrich
Risk Engine fetches EPSS scores → calculates risk per asset → drafts exec summary
7
Alert & Report
Alerts sent via Slack/Email → PDF/CSV/JSON generated → Dashboard shows results
